# 基于加速度传感器的人体活动识别（AIoT方向）

## 项目简介
利用手机加速度传感器采集多场景数据（桌面静止、手中静止、手中行走、口袋行走），通过特征工程和机器学习实现“行走”与“静止”二分类。

## 技术栈
- Python 3.13
- Pandas / NumPy
- Scikit-learn (Random Forest)

## 项目流程
1. 数据采集：手机加速度计（三轴XYZ）
2. 信号处理：合成加速度（消除方向干扰）
3. 特征提取：滑窗 + 7个时域特征（均值、标准差、极差、四分位距、平均变化率等）
4. 模型训练：随机森林（Random Forest）
5. 模型评估：测试集准确率 **90%**

## 核心亮点
- 混合多组真实场景数据（桌面静止、手中静止、手中行走、口袋行走），验证特征方案的鲁棒性
- 三轴合成加速度，消除手机朝向对识别结果的影响
- 纯CPU推理，适合边缘端部署
